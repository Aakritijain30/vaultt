# VAULTT — Premium E-Commerce Store

A fully responsive, multi-page e-commerce website built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

<🔗 [View on GitHub Pages](https://Aakritijain30.github.io/vaultt)

## 🚀 Features

### Pages
| Page | Description |
|------|-------------|
| 🏠 **Home** | Hero section, categories, featured products, promo banner, testimonials, stats |
| 🛍️ **Shop** | Full product listing with filters, search, sorting |
| 📬 **Contact** | Contact form with success state, info cards, FAQ accordion |

### Functionality
- 🛒 **Persistent Cart** — Add, remove, update quantity; saved in `localStorage`
- 🔍 **Live Search** — Filter products by name/description in real-time
- 🎛️ **Filters** — Filter by category, max price (range slider), and minimum rating
- ↕️ **Sorting** — Sort by featured, price (low/high), rating, name A–Z
- 🪟 **Product Modal** — Click any product to view full details
- 📱 **Fully Responsive** — Mobile menu, stacked layouts, touch-friendly
- ✨ **Animations** — Blob animations, marquee, scroll reveal, card transitions
- 🍞 **Toast Notifications** — Shown when items are added to cart
